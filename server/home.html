Administration Panel

This is the administration panel for the Aanganwadi application. Consider the following document a brief tutorial. The main navigation menu has tabs which activate different parts of the panel. We will describe them next.

The administration panel requires you to login with an email - password combination which is the admin user-password combination created during database setup.

Forms

This tab contains all information about forms.

On the left side you have a list of forms that are available. You can click on a form to open it for editing.

The right panelcontains tools for editing the form along with a few form actions.

Content

Content in the application can be controlled through the content panel. The left shows the hierarchy that the app contains.

Clicking on the cross will delete a content and clicking on the content itslef will open it.

A new category can be added using the form provided in the left panel. Simply type the name and click the greren + button. The category gets added into the current location you are viewing.

The actual files can be uploaded Via the content upload form. In case no title is provided, the content's filename is assumed to be the title. Multiple files can be selected for upload.

Content which has been uploaded can be reorganized by uploading an excel file. The file has to be in the proper format. You can upload a file in DoIT&C format or in the generic format. The generic format is as follows:

  • First column must be 'filehash'. This can be left empty
  • Second column must be 'Filename'. This column must contain the filename of the uploaded file.
  • Consequent columns are utilized as meta data about the file. They are available whenever content is requested.
  • After meta data a column must be created with the name 'TreeColumnsMarker'. This can be an empty column. It demarcates the start of the tree path that this file must exist in.
  • All columns subsequent to the 'TreeColumnsMarker' column are treated as a path in a tree. The content is placed at the end of the tree.

Reports

Reports can be obtained by selecting an ending date and a lookback period. The kind of report can also be selected.

Clicking the "Get Reports" button obtains the reports and puts them in a table.
