Help

Home

The home page shows Activity Bank. On opening an activity, the categories of the Activity are shown. On going further, the content associated with the activity is shown. Currently only MP4, MP3 and PDF files are supported for viewing in app.

Forms to Fill

The forms that are available to fill will be listed here. If a user has already filled the form, the same form will no longer be available. If there are no forms to fill, you will be automatically redirected to Home. To view forms a user must be logged in, hence if the user is not logged in and tries to view a form, the user will be automatically redirected to login page.
